简介

  • DB：Database（数据库）

  • DBMS： Database Management System（数据库管理系统 常见：oracle ==mysql== sqlserver……)

  • SQL：结构化查询语言（sql语句执行时都会编译，由DBMS完成）

  • DBMS –>（执行）–> SQL –> （操作） –> DB

1、表（Table）

表由俩个部分组成，字段和数据。行为数据/记录，列为字段。

每个字段都应该包括的属性：字段名，数据类型，相关的约束。

2、SQL语句

  • DQL（数据查询语言）：查询语句，凡是select语句都是DQL.
  • DCL（数据操作语言）： insert delete update，对标当中的数据进行增删改。
  • DDL（数据定义语言）： create drop alter， 对表结构的增删改。
  • TCL（事务控制语言）： commit（提交事务）rollback（回滚事务）
  • DCL（数据控制语言）： grant(授权） revoke（撤销权限）
